registration

アカウントの新規作成を行います。

引数

email:str
メールアドレス
password:str
パスワード
大文字小文字数字の8文字以上14文字以下で構成された文字列であること。
name:str
名前
furigana:str
ふりがな
sex:int
性別
birthday:date
誕生日
residence:str
居住地
県のみ日本語漢字で
qualification:str[]
資格

戻り値

{
"status": bool
アカウントの作成が成功した場合にtrueが返り、失敗したときにfalseが返ります。
"result": str
何らかの情報を返す必要性が生じた場合を想定し、一応用意していますが、現状あまり意味がないただのメッセージを返します。
}
