API manual

APIサーバー/api:3000に対して、フォームデータ形式のPOSTメソッドを送信することでAPIを利用できます。

key:requesのvalueに対し、任意の利用したいAPIを指定してください。
利用できるAPIは以下の通りです。

authentication

メールアドレスとパスワードからアカウントの利用可否を判定します。

registration

アカウントの登録を行います。

user_profile

アカウントの情報を取得します。

company_profile

企業の情報を取得します。
