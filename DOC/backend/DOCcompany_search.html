company_profile

引数

subject:str
検索対象
以下のものから一つを選択 name

code
企業コードから検索
category
業種から検索
office
事業所から検索
qualification
資格から検索

戻り値

{
"status": bool,
"result": [{
"id": ind,

"name": str,

"$subject": str

],
以下検索結果に合致した数繰り返し
}
